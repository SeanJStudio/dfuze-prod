FAQ

  • Supported files

    accdb, csv, cert-1, dat, dbf, mdb, txt, xls, xlsm, xlsx.

  • What is Dfuze?

    Dfuze is an application that simplifies the process of working with large, unstructured data sets. As you may know, working with large amounts of data can be a time-consuming and error-prone process. Traditionally, this has involved running multiple queries in Access or using complex Excel formulas to manipulate the data. Dfuze offers a more efficient and user-friendly solution.

  • What hardware do I need?

    You will need a Windows OS with Java SDK 9.0 or later installed.
